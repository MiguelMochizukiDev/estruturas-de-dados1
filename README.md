# Estruturas de Dados 1

Este repositório contém notas de aula e resoluções de exercícios relativos à disciplina
"Estruturas de Dados I", ministrada na UFPB em 2026.1 pela professora Liliane dos Santos Machado.

## Estrutura do projeto

```bash
estruturas-de-dados1
├── aulas
│   └── 2026-04
│       └── 2026-04-27
│           ├── exercicios
│           │   └── exercicio01.c
│           └── resumo.md
├── include
├── LICENSE
├── Makefile
├── README.md
└── src
```

## Compilando soluções

Resoluções dos exercícios e códigos em `src/` são todos compilados mediante o comando

```bash
make
```

Isso gera dois folders: `build/`, com os arquivos objeto, e `bin/`, com os binários gerados.

Para limpar o diretório, use o comando

```bash
make clean
```
